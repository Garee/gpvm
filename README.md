VM
==

VM is a parallel virtual machine implemented on a GPU using OpenCL.

Install
-------
1. Clone the repository.
2. Compile the source files.
<pre>
make
</pre>
3. Execute the program.
<pre>
./vm
</pre>

Tests
-----
Unit tests are located in the tests/ directory. 

Run all unit tests:

<pre>
make test
</pre>
